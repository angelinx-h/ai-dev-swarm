# gemini-cli Installation

Google's Gemini CLI tool for AI-assisted coding.

## Installation Methods

### macOS, Linux

**Option 1: npm**
```bash
npm install -g @google/gemini-cli
```

**Option 2: Homebrew**
```bash
brew install gemini-cli
```

## Launching

```bash
gemini
```

## Verification

```bash
gemini --version
```

## First-Time Setup

1. Run `gemini` in your terminal
2. Follow the authentication prompts
3. Sign in with your Google account
4. Configure API key if required
