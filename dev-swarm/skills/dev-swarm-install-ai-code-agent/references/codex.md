# codex Installation

OpenAI's Codex CLI tool for AI-assisted coding.

## Installation Methods

### macOS, Linux

**Option 1: npm**
```bash
npm install -g @openai/codex
```

**Option 2: Homebrew**
```bash
brew install codex
```

## Launching

```bash
codex
```

## Verification

```bash
codex --version
```

## First-Time Setup

1. Run `codex` in your terminal
2. Follow the authentication prompts
3. Configure your OpenAI API key
4. Accept terms of service if prompted
